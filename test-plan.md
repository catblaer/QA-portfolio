# Тест-план: Тестирование интернет-магазина (v1.0)

## 1. Введение
Цель данного тестирования — проверить работоспособность основных функций (регистрация, логин, корзина) и убедиться, что пользователь может успешно совершить покупку.

## 2. Объект тестирования
Веб-приложение "Sample E-commerce Store".

## 3. Область тестирования (Scope)
- **Функциональное тестирование:** Регистрация, Авторизация, Поиск товара, Добавление/Удаление из корзины.
- **UI тестирование:** Проверка адаптивности на мобильных устройствах.
- **Тестирование валидации:** Поля ввода (email, пароль).

## 4. Инструменты
- Браузеры: Chrome (десктоп), Safari (мобильный).
- Инструменты разработчика: Chrome DevTools.

## 5. Критерии начала и окончания
- **Начало:** Наличие стабильной версии сайта и требований.
- **Окончание:** Выполнены все запланированные тест-кейсы, исправлены критические баги (Critical/Blocker).

## 6. Оценка рисков
- Отсутствие доступа к базе данных (тестирование только "черным ящиком").
- Сжатые сроки тестирования.
